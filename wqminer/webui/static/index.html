WQMiner WebUI

低占用的本地管理界面，分离字段拉取、模板生成与优化循环。

字段拉取 灵感与模板 提交-反馈-优化 记录查看 任务列表

工作流拆分

每个功能都是独立任务：拉字段、生成模板、提交反馈优化。任务会在后台线程中运行，避免阻塞 UI。

低占用原则

默认单线程任务队列，UI 仅轮询结果。可通过环境变量调整并发与端口。

一键自动优化

流程：模板生成一次 → 覆盖字段回测 → 每轮基于上一轮数据单次进化 → 持续回测。

高级参数 

        
如需停止任务，请到“任务列表”页面输入 Job ID 终止。
